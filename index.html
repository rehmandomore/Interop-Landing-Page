What is interop?

Interop is a single global account that aggregates all the chains to a single network, wherein users have the ability to choose which network they’d like to process the transaction. Instead of having multiple networks and their native tokens, there is only a single native gas token INST, meaning you can make a transaction on Polygon using INST instead of MATIC, and so on.

Does Interop charge any fees?

Since the native gas on Interop is “INST”, the validators pay the transaction gas for the user and deduct 1.2x the INST as gas. This is deducted directly from their Interop INST balance.
